# 飞书自定义公式指标卡插件

这是一个飞书多维表格仪表盘插件，支持自定义加权平均公式计算指标。

## 功能特性

- 支持自定义公式：`SUM(分子) / SUM(分母)`
- 支持数值、百分比、货币三种格式
- 可自定义前缀、后缀、小数位数
- 完美还原飞书原生指标卡样式

## 快速开始

### 1. 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 2. 构建部署

```bash
# 构建生产版本
npm run build

# 部署到 Vercel
vercel --prod
```

## 使用说明

1. 在飞书多维表格中创建仪表盘
2. 添加自定义插件，输入插件 URL
3. 点击插件右上角齿轮图标配置
4. 选择分子字段、分母字段、数值格式
5. 保存配置即可显示计算结果

## 公式说明

插件会自动计算：
```
指标值 = SUM(分子字段的所有记录) / SUM(分母字段的所有记录)
```

适用于计算平均单价、平均成本、转化率等加权平均值。
